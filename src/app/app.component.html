About AirThin Currency Convert

AirThin Currency Convert is a Lightweight Currency Converter WebApp that makes use of the ExchangeRate API in order to fetch correct Currency Rates. The rates are updated every day to ensure consistency.

This project was built using Angular 9 and Materialize-CSS
